# ⛳️ React 개인 프로젝트 - 말본 골프(Malbon Golf) 클론

React, Redux Toolkit, React-Router를 기반으로 구현한 프리미엄 골프웨어 쇼핑몰 웹 프로젝트입니다.

## 📌 목차
- 개요
- 기술 스택
- 기능 구현 PPT
- 개선사항 (회고)

---

## 🗂 개요
- **프로젝트 목표:** 말본 골프의 감각적인 UI 완벽 구현 및 이커머스 핵심 비즈니스 로직(동적 검색, 라우팅, 장바구니, 리뷰 게시판) 개발
- **개발 인원:** 1인 (프론트엔드 개인 프로젝트)

## 🛠 기술 스택
- **Language:** JavaScript (ES6+)
- **Framework / Library:** React, Redux Toolkit, React Router DOM
- **Styling:** CSS3 (순수 CSS), Swiper
- **Data:** Local JSON (Mock Data 비동기 호출)
- **Tool:** VS Code, Figma

## 🚀 개선사항 및 회고 (추후 업데이트 예정)
1. **반응형 웹 적용:** 현재 데스크탑 기준의 UI를 모바일 및 태블릿 환경에서도 최적화되도록 미디어 쿼리(Media Query) 적용 예정입니다.
2. **성능 최적화:** 이미지 지연 로딩(Lazy Loading) 및 코드 스플리팅을 도입하여 초기 렌더링 속도와 성능을 개선할 예정입니다.
